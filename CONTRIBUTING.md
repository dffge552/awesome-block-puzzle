# Contribution Guidelines

Thank you for considering contributing to Awesome Block Puzzle! Please ensure your pull request adheres to the following guidelines:

## Adding to this list

- **Search previous suggestions** before making a new one to avoid duplicates.
- **Make an individual pull request** for each suggestion.
- Use the following format: `[Project Name](link) - Description.`
- Descriptions should be clear, concise, and end with a period.
- New categories or improvements to the existing categorization are welcome.
- Check your spelling and grammar.
- Make sure your text editor is set to remove trailing whitespace.

## Quality Standards

Your suggestion should meet these criteria:

- **Relevance**: Must be a grid-based placement puzzle game (8√ó8, 9√ó9, 10√ó10, etc.) with row/column elimination.
- **Active OR Historical**: Either actively maintained OR has significant historical/educational value.
- **Working Links**: All links must be functional and point to legitimate resources.
- **Description**: Must have a clear, informative description explaining what makes it awesome.
- **Not Tetris**: Do not submit Tetris-style falling block games (they belong in [Awesome Tetris](https://github.com/omercevik/awesome-tetris)).

## Updating your PR

If the maintainers notice anything that should be changed, they will ask you to edit your PR before merging it. There's no need to open a new PR, just edit the existing one. If you're not sure how to do that, [here is a guide](https://github.com/RichardLitt/knowledge/blob/master/github/amending-a-commit-guide.md) on the different ways you can update your PR.

## Pull Request Process

1. Fork the repository.
2. Add your link in the appropriate category.
3. Ensure the link works and follows the format above.
4. Run `npx awesome-lint` to check for errors.
5. Commit with a descriptive message like: `Add [Project Name]`.
6. Create a pull request with a clear title: `Add [Project Name]`.

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.
