## What is Block Puzzle?

Block Puzzle is a family of grid-based puzzle games where:
- ‚úÖ Players place shapes on a fixed grid (8√ó8, 9√ó9, or 10√ó10)
- ‚úÖ Complete **rows OR columns** are cleared (unlike Tetris which only clears rows)
- ‚úÖ Turn-based gameplay with no time pressure
- ‚úÖ Strategy over reflexes

**Popular variants:**
- **Block Blast** (8√ó8) - Mobile game with combo system
- **1010!** (10√ó10) - Classic variant with color scoring
- **Woodoku/Blockudoku** (9√ó9) - Combines Sudoku 3√ó3 region rules

**Key differences from Tetris:**

| Feature | Block Puzzle | Tetris |
|---------|--------------|--------|
| Gameplay | Free placement | Falling blocks |
| Grid | Static | Dynamic |
| Clear rule | Rows **OR** columns | Rows only |
| Timing | Turn-based | Real-time |
